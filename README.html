サービス利用規約（SNS）

この利用規約（以下、「本規約」という）は、鈴木杏南（以下、「当社」という）が提供するAscentry（以下「本サービス」という）の利用条件を定めるものです。利用者の皆さまには、本規約に従って、本サービスをご利用いただきます。

第1条（適用）

1. 本サービスを利用するには、利用規約の全文をお読みいただいたうえで、本規約に同意いただく必要があります。

2. 本規約は、利用者と当社との間の本サービスの利用に関わる一切の関係に適用されるものとします。

3. 本サービスの利用に当たっては、本規約の他、当社が別途定めるプライバシーポリシー（https://doc-hosting.flycricket.io/ascentry-privacy-policy/56e6ce84-a208-469e-9405-0021839404c1/privacy）等の個別規定が適用されます。

第2条（用語の定義）

  • 「本サービス」：当社が提供する「Ascentry」およびそれに付随する一切のサービス
  • 「利用者」：本規約に同意のうえ、本サービスを利用する者
  • 「利用者コンテンツ」：利用者が投稿する文章、画像、音声等

第3条（利用登録）

1. 利用希望者は、本規約に同意のうえ利用登録を行い、当社が承認した時点で契約が成立します。

2. 正確な情報を登録するものとします。

第4条（認証情報の管理）

利用者は、IDやパスワードを自己責任で管理するものとします。

第5条（利用環境）

必要な通信環境は利用者の責任で準備するものとします。

第6条（利用料金）

本サービスは無料で利用できます。

第7条（知的財産権）

本サービスに関する権利は当社または正当な権利者に帰属します。

第8条（サービスの停止）

保守や障害時にサービスを停止する場合があります。

第9条（サービスの変更・終了）

当社はサービス内容を変更・終了することがあります。

第10条（禁止事項）

第17条（規約変更）
